Eingereichte Erfahrungsberichte sichten und freigeben

In diesem Bereich können Sie die eingereichten Erfahrungsberichte sichten, den Autor_innen eine Rückmeldung geben und freigeben.

Nachfolgend finden Sie alle Erfahrungsberichte aufgelistet, die aktuell von Autor_innen zur Sichtung eingereicht wurden. Falls Sie Rückfragen zum eingereichten Erfahrungsberichte haben, können Sie die Autor_innen direkt über die Plattform kontaktieren.

Sofern die Sichtung des eingereichten Erfahrungsberichte positiv ausfällt, können Sie das Element im Editiermodus (blauer Button) "freigeben". Der Erfahrungsberichte wird anschließend direkt live im digital.learning.lab anzeigt. Sofern die Sichtung des eingereichten Erfahrungsberichte negativ ausfällt, können Sie das Element im Editiermodus "zur Nachbearbeitung" an die Autor_innen zurückschicken. Der Erfahrungsberichte wird dadurch wieder für die Autor_innen zur Bearbeitung freigeschaltet und erscheint nicht live im digital.learning.lab.
